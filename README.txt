使用方法：
1. 请先修改scanner.xml文件， 设置引擎的路径
2. cd croscan目录
3. 启动croscan
	a. 如果是想测试或者扫描个别样本，可以使用交互模式
		croscan -i
		然后根据提示输入样本的全路径
	b. 如果是想要持续不断地扫描大量样本， 可以使用管道模式
		croscan -p <管道名>
	   然后在管道的另外一端的程序可以不断地向管道中输入样本路径
	   （sender.exe的源码是一个样例， 可以参考）
4. 在croscan/log目录下查看扫描结果
